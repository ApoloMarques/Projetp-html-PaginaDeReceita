# Página de Receita – Projeto HTML

## 📌 Descrição

Projeto web desenvolvido para apresentação de uma página de receita culinária utilizando estrutura HTML, estilização em CSS e organização visual responsiva.

O objetivo do projeto é praticar fundamentos de desenvolvimento front-end, estruturação de conteúdo e criação de interfaces simples e intuitivas.

---

## 🌐 Acesse o Projeto

🔗 https://apolomarques.github.io/projetp-html-pagina-de-receita/

---

## 🎯 Objetivos

- Praticar estruturação de páginas em HTML
- Desenvolver estilização com CSS
- Criar layout organizado e responsivo
- Melhorar habilidades em desenvolvimento front-end

---

## 💡 Funcionalidades
